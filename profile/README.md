# SSAFY-B102-EyeDontCare
SSAFY 2학기 공통 프로젝트

## 프로젝트 소개
업무에서 완전히 이탈하지 않으면서, 동료들과 잠깐 한눈팔며 리프레시하는 눈 기반  소셜 마이크로 브레이크 서비스

## 서비스 설명/주요 기능
1. 실시간 P2P 1:1 눈싸움 대결
2. 실시간 승패 및 부정 판정
3. 누적 전적 및 랭킹 시스템
4. 플레이어 성향 프로필 카드 발급
5. 소모임 관리
